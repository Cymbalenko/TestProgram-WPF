Мне не понравилось ни одно из экзаменационных заданий, поэтому я сделал свою тему!
Создал приложений «Система для создания тестов и их прохождение».

Приложение на основе паттерна MVVM
Работа с базой с помощью Entity Framework с подходом Code First 
Поставил перед собой такие задачи:
¦¦Добавление\редактирование\удаление теста который состоит из:
  1)Темы также Добавление\редактирование\удаление
  2)Вопросов также Добавление\редактирование\удаление
  3)Ответов также Добавление\редактирование\удаление

¦¦Добавить возможность в вопросы добавлять не только текст, а и звук и картинки,
для того что-бы можно было создавать\редактировать любые тесты.
(Все добавленные файлы копируються в папки проекта(Images и Audio)

¦¦Добавить такие настройки перед запуском теста как:
  1)Запустить все тесты которые доступны в выбранной теме.(Так же возможность запустить рандомно)
  2)Запустить 'n'кол-во тестов из всей доступной выборки в выбранной теме.(Так же возможность запустить рандомно)
  3)Запустить с 'n1' по 'n2' тестов.(Так же возможность запустить рандомно)
  4)Возможность запустить тест на время(И ввод кол-во минут на прохождение).
  5)Ввести имя студента который проходит тест  
  6)Выбор показывать\непоказывать кол-во правельных ответов.
  7)Выбор включить\выключить возможность возвращатся к пройденным вопросам.
  8)Выбор показывать\непоказывать правельный вариант ответа.

¦¦Добавить историю пройденных тестов.

¦¦Окно с результатом пройденного теста(появляется после теста).

¦¦Добавить визуал.

¦¦Добавить загрузку с файла с помощью трёх сеппараторов (
  1-Чтобы отделить вопросы между собой
  2-Чтобы отделить вопрос, от вариантов ответа
  3-Чтобы отделить варианты ответа друг от друга)
